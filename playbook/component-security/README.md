# Component Security

## Overview
Component security focuses on protecting the individual hardware and software components that make up a computer system. If a single component is compromised, it can affect the security of the entire system.

## Topics
- Operating Systems
- Hardware Security
- Firmware
- BIOS / UEFI
- Trusted Platform Module (TPM)
- Secure Boot
- Device Drivers
- Peripheral Devices
- Virtual Machines
- Endpoint Security

## Skills
- Hardening operating systems
- Securing hardware and firmware
- Managing device configurations
- Applying security patches
- Protecting endpoints
