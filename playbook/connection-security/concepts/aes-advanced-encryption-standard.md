# Advanced Encryption Standard (AES)

## Overview

Advanced Encryption Standard (AES) is the most widely used symmetric encryption algorithm. It uses the same secret key to encrypt and decrypt data and is trusted worldwide to protect sensitive information.

AES is the current encryption standard and has replaced older algorithms such as DES.

## Key Concepts

- Symmetric Encryption
- Shared Secret Key
- AES-128
- AES-192
- AES-256
- Block Cipher

## Why It Matters

AES provides fast, strong encryption for protecting data at rest and in transit. It is the primary algorithm used whenever large amounts of data need to be encrypted efficiently.

## Common Uses

- Full Disk Encryption
- File Encryption
- HTTPS
- TLS
- VPNs
- Wi-Fi (WPA2/WPA3)

## Best Practices

- Use AES-256 when appropriate for maximum security.
- Protect the secret encryption key.
- Generate strong, random keys.
- Pair AES with secure key exchange methods such as Diffie-Hellman or RSA.
