# Denial-of-Service (DoS)

## Overview

A Denial-of-Service (DoS) attack attempts to make a system or service unavailable by overwhelming it with requests or exploiting a vulnerability until legitimate users can no longer access it.

## Key Concepts

- Service disruption
- Resource exhaustion
- Single attack source
- Availability
- Network flooding

## Why It Matters

Availability is one of the core principles of the CIA Triad. DoS attacks target this principle by preventing legitimate access to systems or services.

## Common Uses

- Website disruption
- Network outages
- Service interruption
- Resource exhaustion

## Best Practices

- Use firewalls and rate limiting.
- Keep systems patched.
- Monitor unusual traffic spikes.
- Deploy DDoS protection services.
