# IPv4 Addressing

## Overview

IPv4 is the most widely used addressing system for computer networks. An IPv4 address is a 32-bit logical address written as four decimal octets separated by periods. Every device on an IPv4 network requires a unique IP address.

## Key Concepts

- 32-bit address
- Four octets
- Network address
- Host address
- Subnet mask
- Default gateway
- Public IP addresses
- Private IP addresses

## Why It Matters

IPv4 addressing is fundamental to networking and cybersecurity. Understanding how IP addresses work is essential for troubleshooting, packet analysis, firewall configuration, routing, vulnerability scanning, and penetration testing.

## Common Uses

- Identifying devices on a network
- Routing traffic
- Network troubleshooting
- Firewall rules
- Network scanning

## Best Practices

- Assign unique IP addresses.
- Understand the difference between public and private addresses.
- Verify subnet masks and default gateways when troubleshooting.
- Never assign duplicate IP addresses.
