# IPv6

## Overview

IPv6 is the successor to IPv4 and was developed to solve IPv4 address exhaustion while improving network scalability and security. IPv6 uses 128-bit addresses written in hexadecimal notation.

## Key Concepts

- 128-bit addresses
- Hexadecimal notation
- Address expansion
- Internet of Things (IoT)
- Native IPv6 support

## Why It Matters

As organizations continue adopting IPv6, security professionals must understand how it works. Many modern operating systems enable IPv6 by default, and attackers may exploit poorly configured IPv6 environments to bypass security controls.

## Common Uses

- Enterprise networking
- Internet communication
- Cloud infrastructure
- IoT devices
- Modern operating systems

## Best Practices

- Learn to recognize IPv6 addresses.
- Disable IPv6 only when appropriate and supported by policy.
- Ensure security devices inspect IPv6 traffic.
- Include IPv6 during security assessments.
