# Ports and Port Numbers

## Overview

Ports identify the specific service or application running on a device. While an IP address identifies the host, a port identifies the process receiving network traffic.

## Key Concepts

- TCP ports
- UDP ports
- Well-known ports (0–1023)
- Registered ports
- Dynamic (ephemeral) ports
- Service identification

## Why It Matters

Open ports reveal what services are available on a system. Security professionals use ports to identify services, configure firewalls, detect unauthorized software, and perform vulnerability assessments.

## Common Uses

- HTTP (80)
- HTTPS (443)
- SSH (22)
- FTP (20/21)
- DNS (53)
- SMTP (25)

## Best Practices

- Disable unused services.
- Close unnecessary ports.
- Restrict access using firewalls.
- Regularly scan systems for unexpected open ports.
