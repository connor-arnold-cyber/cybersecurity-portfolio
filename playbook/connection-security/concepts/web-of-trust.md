# Web of Trust

## Overview

A Web of Trust is a decentralized trust model in which users verify and trust each other's public keys instead of relying on a centralized Certificate Authority (CA). Trust is established through individuals digitally signing one another's public keys.

The Web of Trust is commonly associated with PGP and GPG.

## Key Concepts

- Trust Relationships
- Public Keys
- Digital Signatures
- Decentralized Trust
- PGP / GPG

## Why It Matters

The Web of Trust provides an alternative to traditional Public Key Infrastructure (PKI) by allowing users to establish trust without depending on a central authority. This makes it useful in environments where centralized trust is impractical.

## Common Uses

- PGP
- GPG
- Secure Email
- Open Source Communities
- Personal Key Verification

## Best Practices

- Verify identities before signing public keys.
- Only trust keys from reputable sources.
- Protect your private key.
- Regularly review trusted keys.
- Revoke compromised keys promptly.
