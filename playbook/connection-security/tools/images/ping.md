# Ping

## Overview

Ping is a network troubleshooting tool that uses ICMP Echo Request and Echo Reply messages to determine whether a remote device is reachable and measure basic network latency.

## Key Concepts

- ICMP Echo Request
- ICMP Echo Reply
- Latency
- Packet loss
- Connectivity testing

## Why It Matters

Ping is one of the first tools used during network troubleshooting. Security professionals also use it during reconnaissance, though many organizations restrict ICMP traffic to reduce information disclosure.

## Common Uses

- Verify network connectivity
- Test whether a host is online
- Measure latency
- Troubleshoot communication problems

## Best Practices

- Verify connectivity before deeper troubleshooting.
- Remember that blocked ICMP does not always mean a host is offline.
- Combine Ping with other diagnostic tools when necessary.
