# Data Security

## Overview
Data security focuses on protecting data throughout its lifecycle by ensuring its confidentiality, integrity, and availability. It includes safeguarding data at rest, in transit, and in use from unauthorized access, alteration, or destruction.

## Core Concepts
- Confidentiality
- Integrity
- Availability (CIA Triad)
- Encryption
- Hashing
- Data Classification
- Access Control
- Digital Signatures
- Backup and Recovery
- Data Loss Prevention (DLP)

## Common Threats
- Data Breaches
- Unauthorized Access
- Data Leakage
- Ransomware
- Insider Threats
- Data Corruption
- Data Theft
- Improper Data Disposal

## Best Practices
- Encrypt sensitive data.
- Apply the principle of least privilege.
- Classify data based on sensitivity.
- Create and regularly test backups.
- Verify data integrity with hashing.
- Securely erase retired storage devices.
- Monitor and audit access to sensitive data.
