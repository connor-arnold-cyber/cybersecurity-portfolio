# AES (Advanced Encryption Standard)

## Overview

Advanced Encryption Standard (AES) is the world's most widely used **symmetric encryption algorithm**. It encrypts data using a shared secret key and provides fast, secure encryption for protecting data at rest and data in transit.

AES replaced DES as the U.S. encryption standard in 2001 and remains the industry standard today.

## Key Concepts

- Symmetric Encryption
- Shared Secret Key
- Block Cipher
- 128-bit Block Size
- AES-128
- AES-192
- AES-256

## Why It Matters

AES is the standard encryption algorithm used to protect sensitive data across modern computing. Its combination of strong security and high performance makes it the foundation of many secure systems and protocols.

## Common Uses

- HTTPS/TLS
- VPNs
- WPA2/WPA3 Wi-Fi
- Full Disk Encryption
- File Encryption
- Password Managers
- Cloud Storage

## Best Practices

- Use AES-256 when supported.
- Protect encryption keys.
- Use secure modes of operation.
- Pair AES with secure key exchange methods.
- Never create your own encryption algorithm.
