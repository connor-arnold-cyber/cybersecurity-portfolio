# Hybrid Encryption

## Overview

Hybrid encryption combines asymmetric and symmetric encryption. Asymmetric encryption securely exchanges a shared secret key, while symmetric encryption uses that key to encrypt the actual data.

Most modern secure communication uses hybrid encryption.

## Key Concepts

- Key Exchange
- Session Key
- Public Key
- Private Key
- [Symmetric Encryption](symmetric-encryption.md)
- [Asymmetric Encryption](asymmetric-encryption.md)

## Why It Matters

Hybrid encryption provides the security of asymmetric encryption and the speed of symmetric encryption, making it practical for real-world communication.

## Common Uses

- HTTPS
- TLS
- VPNs
- Secure Messaging
- Online Banking

## Best Practices

- Use asymmetric encryption only to exchange keys.
- Use symmetric encryption for data transmission.
- Generate a unique session key for each connection.
