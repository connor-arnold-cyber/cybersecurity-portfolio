# RSA

## Overview

RSA is one of the most widely used asymmetric encryption algorithms. It uses a public key to encrypt data and a private key to decrypt it. RSA is also commonly used to create and verify digital signatures.

## Key Concepts

- Public Key
- Private Key
- 2048-bit Keys
- 4096-bit Keys
- Digital Signatures

## Why It Matters

RSA provides secure key exchange, authentication, and digital signatures. It remains one of the most common public key algorithms used today.

## Common Uses

- TLS
- HTTPS
- Digital Certificates
- Email Encryption
- Code Signing

## Best Practices

- Use key lengths of at least 2048 bits.
- Protect private keys.
- Replace compromised keys immediately.
- Use RSA alongside symmetric encryption.
