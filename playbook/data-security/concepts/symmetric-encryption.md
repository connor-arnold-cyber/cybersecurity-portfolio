# Symmetric Encryption

## Overview

Symmetric encryption is a cryptographic method that uses the same secret key to both encrypt and decrypt data. Because both parties must possess the same key, securely sharing that key is the biggest challenge.

Symmetric encryption is fast and is commonly used to encrypt large amounts of data.

## Key Concepts

- Shared Secret Key
- Plaintext
- Ciphertext
- AES
- DES

## Why It Matters

Symmetric encryption provides fast, efficient confidentiality. It is used by nearly every modern encryption system once a secure session has been established.

## Common Uses

- Full Disk Encryption
- File Encryption
- VPNs
- TLS Sessions
- Wi-Fi Encryption

## Best Practices

- Protect the secret key.
- Use modern algorithms such as AES.
- Never share the key over an insecure channel.
- Rotate keys when appropriate.
