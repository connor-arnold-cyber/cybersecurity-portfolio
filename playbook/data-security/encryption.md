# Encryption

## Overview

Encryption is the process of converting readable data (plaintext) into unreadable data (ciphertext) using a cryptographic algorithm and a key. Only someone with the correct key can decrypt the ciphertext back into plaintext.

Encryption is primarily used to protect the confidentiality of data.

## Key Concepts

- Plaintext
- Ciphertext
- Encryption Key
- Decryption Key
- Symmetric Encryption
- Asymmetric Encryption

## Why It Matters

Encryption protects sensitive information from unauthorized access. Even if encrypted data is intercepted or stolen, it remains unreadable without the correct decryption key.

## Common Uses

- HTTPS
- VPNs
- Full Disk Encryption (BitLocker)
- Email Encryption
- File Encryption

## Best Practices

- Use modern encryption algorithms.
- Protect encryption keys.
- Encrypt sensitive data both at rest and in transit.
- Replace deprecated encryption standards.
