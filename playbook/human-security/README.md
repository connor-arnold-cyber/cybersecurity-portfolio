# Human Security

## Overview
Human security focuses on protecting people from becoming the weakest link in cybersecurity. It emphasizes security awareness, identity protection, and reducing the risk of human error, social engineering, and insider threats.

- Social Engineering

## Best Practices
- Use strong, unique passwords.
- Enable multi-factor authentication (MFA).
- Verify requests before sharing sensitive information.
- Complete regular security awareness training.
- Follow the principle of least privilege.
- Report suspicious activity immediately.
- Protect personal and organizational credentials.
