# Adware

## Overview

Adware is software that displays advertisements and often collects information about user behavior to deliver targeted advertising. Some forms of adware are considered potentially unwanted programs (PUPs).

## Key Concepts

- Advertising
- Tracking
- Browser redirects
- User profiling
- Privacy

## Why It Matters

While often less destructive than other malware, adware can reduce system performance, invade privacy, and introduce additional security risks.

## Common Uses

- Displaying advertisements
- Tracking user activity
- Redirecting web traffic
- Collecting marketing data

## Best Practices

- Install software from trusted sources.
- Review installation options carefully.
- Use reputable anti-malware software.
- Remove unnecessary browser extensions.
