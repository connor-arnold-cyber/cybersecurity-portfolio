# Backdoor

## Overview

A backdoor is a hidden method of bypassing normal authentication to gain future access to a system. Backdoors may be intentionally installed by attackers or included within malware.

## Key Concepts

- Unauthorized access
- Persistence
- Remote access
- Authentication bypass
- Command and control

## Why It Matters

Backdoors allow attackers to return to compromised systems without exploiting the original vulnerability again, making them a major concern during incident response.

## Common Uses

- Persistent remote access
- Malware communication
- Command execution
- Data exfiltration

## Best Practices

- Remove unauthorized accounts and services.
- Monitor outbound connections.
- Perform integrity checks after incidents.
- Reimage compromised systems when appropriate.
