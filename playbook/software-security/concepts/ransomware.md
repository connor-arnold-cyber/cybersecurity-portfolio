# Ransomware

## Overview

Ransomware is malware that encrypts files or systems and demands payment in exchange for restoring access. Many ransomware attacks also steal data before encryption.

## Key Concepts

- Encryption
- Extortion
- Data recovery
- Double extortion
- Backup strategy

## Why It Matters

Ransomware is one of the most financially damaging cyber threats facing organizations today. Recovery is often expensive and time-consuming.

## Common Uses

- Encrypting files
- Stealing sensitive data
- Extorting organizations
- Disrupting business operations

## Best Practices

- Maintain offline backups.
- Patch vulnerabilities.
- Restrict user permissions.
- Implement multi-factor authentication.
- Educate users about phishing.
