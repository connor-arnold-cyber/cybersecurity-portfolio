# Trojan

## Overview

A Trojan is malware disguised as a legitimate program. Unlike viruses and worms, Trojans rely on users to install or execute them.

## Key Concepts

- Disguised software
- Social engineering
- Backdoor
- Remote access
- Payload

## Why It Matters

Trojans often provide attackers with persistent access to compromised systems. They are frequently delivered through phishing emails and fake software downloads.

## Common Uses

- Remote access
- Credential theft
- Installing malware
- Data exfiltration

## Best Practices

- Download software only from trusted sources.
- Verify digital signatures when possible.
- Train users to recognize phishing attempts.
- Use application whitelisting.
