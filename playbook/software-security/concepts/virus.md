# Virus

## Overview

A virus is malware that attaches itself to a legitimate file or program. It requires user action, such as opening an infected file, to spread to other systems.

## Key Concepts

- Host file
- User interaction
- Replication
- Payload
- Infection

## Why It Matters

Viruses remain a common method of compromising systems through email attachments, downloads, and removable media. Understanding how viruses spread helps security professionals reduce infection risk.

## Common Uses

- Corrupting files
- Stealing information
- Destroying data
- Installing additional malware

## Best Practices

- Scan downloaded files.
- Disable unnecessary macros.
- Keep antivirus definitions current.
- Avoid opening suspicious attachments.
