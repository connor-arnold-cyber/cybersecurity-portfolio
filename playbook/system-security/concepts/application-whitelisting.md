# Application Whitelisting

## Overview

Application whitelisting is a security control that allows only approved applications to run on a system. Any application not explicitly approved is blocked by default.

## Key Concepts

- Allow list
- Approved applications
- Least privilege
- Malware prevention
- Endpoint protection

## Why It Matters

Application whitelisting helps prevent malware, ransomware, and unauthorized software from executing, even if a user accidentally downloads a malicious file.

## Common Uses

- Enterprise endpoint protection
- Kiosks
- Critical infrastructure
- High-security environments

## Best Practices

- Maintain an up-to-date whitelist.
- Review approved applications regularly.
- Combine with endpoint protection and least privilege.
- Test updates before deployment.
