# Projects

This folder contains my best and most polished cybersecurity work.

Each project is designed to demonstrate practical, real-world skills relevant to a SOC Analyst role, including analysis, problem-solving, and security-focused thinking.

## Project Types

- Network analysis
- Incident response
- Log analysis
- Security-focused problem solving

## Approach

Projects are built from hands-on labs, coursework, and independent practice, then expanded beyond their original scope and refined to demonstrate real-world cybersecurity skills.

This includes:
- Deeper explanations and context
- Step-by-step technical breakdowns
- Supporting screenshots and evidence
- Real-world security context

## Goal

Showcase the ability to take foundational knowledge and turn it into meaningful, job-relevant cybersecurity work.
